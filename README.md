# 📁 Shrishti Tomar — Professional Developer Portfolio

**Live Link:** [https://my-portfolio-beta-steel-ih418ur0sz.vercel.app/](https://my-portfolio-beta-steel-ih418ur0sz.vercel.app/)

A high-performance, responsive personal portfolio website built with Next.js (App Router), React, Framer Motion, and Tailwind CSS. The design uses a premium, modern Indigo-Cyan tech blueprint aesthetic featuring clean grids and custom micro-animations.

---

## ✨ Key Features

* **💻 Minimalist & Modern Design:** Responsive blueprint layout styled with custom Indigo-Cyan accents, grid details, and glowing neon nodes.
* **🤖 Interactive System Diagram:** A custom-styled SVG node architecture representing key focus areas (Frontend, Backend, Database, AI Agents) with center core initials `ST`.
* **📖 Biography & Story Section:** Detailed showcase of B.Tech CSE details, hackathon wins, technical skills, and certifications.
* **📂 Projects Showcase:** A filterable grid containing features, stacks, codes, and live demo links of key projects (RepoInsight, PrepWise, DropAlert, and AI Cold Email Generator).
* **📬 Integrated Contact Form:** Seamless client-side message delivery powered by pre-filled `mailto:` client configurations.
* **🚀 Production Built:** Optimized production routing and assets bundling out of the box with zero environment variables needed.

---

## 🛠️ Tech Stack

* **Framework:** Next.js (App Router)
* **Components:** React (Functional Components, Hooks)
* **Styling:** Tailwind CSS (Custom HSL theme system)
* **Animations:** Framer Motion (Transitions, Hover flips, Fade-ins)
* **Icons:** Lucide Icons
* **Deployment:** Hosted on Vercel

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- Git installed

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/shrish98/my-portfolio.git
   cd my-portfolio
   ```

2. **Install all dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser to view the local server.

4. **Build the production package:**
   ```bash
   npm run build
   ```
